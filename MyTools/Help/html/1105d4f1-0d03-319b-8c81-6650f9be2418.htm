BDConnection Méthodes

Le type BDConnection expose les membres suivants.

Méthodes
  NomDescription
AdministrateBDD
Exécute une requête d'administration de base de données (INSERT, UPDATE, DELETE).
Finalize
Destructeur appelé dès qu'il n'y a plus de référence sur un objet donné, ou dans n'importe quel ordre pendant la séquence d'arrêt.
(Overrides ObjectFinalize.)
GenerateConnectionString
Génère la chaîne de connexion à la base de données.
GetBDConnection
Fonction statique qui crée l'unique instance de la classe BDConnexion.
GetDataTable
Création d'une DataTable en fonction du résultat d'une requête stockée dans un curseur MySqlDataReader.
GetNewMySqlCommand
Génère une nouvelle instance de la classe MySqlCommand commande pour pouvoir manipuler une base de données (contient la requête et la connexion correspondante).
InitConnection
Initialise une nouvelle connexion à la base de données.
ReqDelete
Permet d'exécuter une requête DELETE.
ReqInsert
Permet d'exécuter une requête INSERT.
ReqSelect
Effectue une requête SELECT sur la base de données puis remplis un Dictionnaire : (clé : nomColonne, valeur : colonneValue).
ReqUpdate
Permet d'exécuter une requête UPDATE.
